# Object Detection for 52 Classes with YOLO

Проект выполнен в рамках соревнования по компьютерному зрению. Задача состояла в том, чтобы обучить модель детекции объектов на ограниченном датасете и сформировать файл `sample_submission.csv` с предсказаниями для тестовых изображений.

## Описание задачи

Необходимо было решить задачу object detection для **52 классов**. Для каждого изображения из тестовой выборки нужно было определить объекты, их классы и координаты bounding boxes.

Формат ответа для `PredictionString`:

```text
class confidence x_min y_min x_max y_max class confidence x_min y_min x_max y_max ...
```

Метрика соревнования — **AP по всем классам с IoU-порогом 0.5**.

## Данные

Датасет состоит из двух архивов с данными и файла-примера для отправки ответа:

- `train.zip` — обучающая выборка;
- `test.zip` — тестовая выборка;
- `sample_submission.csv` — пример файла для сабмита.

Структура обучающей выборки:

```text
train/
├── images/   # изображения обучающего датасета
└── labels/   # разметка в формате YOLO
```

Структура тестовой выборки:

```text
test/
└── images/   # изображения тестового датасета
```

## Что было сделано

В ноутбуке `solution.ipynb` реализован полный пайплайн решения:

1. Загружены и распакованы архивы с train/test-данными.
2. Зафиксирован seed `993` для воспроизводимости результата.
3. Проведен EDA:
   - подсчитано количество train/test-изображений;
   - проверено количество классов;
   - изучено распределение объектов по классам;
   - проанализировано количество объектов на изображение;
   - проверены размеры изображений;
   - визуализированы примеры изображений с bounding boxes.
4. Подготовлена структура данных для обучения YOLO.
5. Обучающая выборка разделена на train/validation.
6. Создан конфигурационный файл `data.yaml` для 52 классов.
7. Обучена модель `YOLO11m`.
8. Выполнен инференс на тестовых изображениях.
9. Сформирован итоговый файл `submission_conf001.csv` в нужном формате.

## Модель

Для решения использовалась модель **YOLO11m** из библиотеки `ultralytics`.

Основные параметры обучения:

- модель: `yolo11m.pt`;
- количество классов: `52`;
- размер изображения: `640`;
- количество эпох: `30`;
- batch size: `4`;
- seed: `993`.

Для инференса использовались параметры:

- `conf = 0.01`;
- `iou = 0.5`;
- `max_det = 300`;
- `augment = True`.

## Результат

На выходе ноутбук формирует файл с предсказаниями:

```text
submission_conf001.csv
```

Файл содержит две колонки:

- `image_id` — название тестового изображения;
- `PredictionString` — предсказанные классы, confidence score и координаты bounding boxes.

Лучший полученный результат на соревновании: **AP@0.5 = 0.14342**.

## Как запустить проект

1. Открыть ноутбук `solution.ipynb` в Google Colab.
2. Включить GPU:  
   `Runtime` → `Change runtime type` → `GPU`.
3. Установить зависимости:

```bash
pip install ultralytics
```

4. Загрузить файлы:

```text
train.zip
test.zip
sample_submission.csv
```

5. Запустить все ячейки ноутбука последовательно.
6. После инференса скачать файл `submission_conf001.csv`.
